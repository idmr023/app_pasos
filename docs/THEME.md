# Tema y Paleta de Colores

## Esquema de Colores

| Color | Hex | Uso |
|-------|-----|-----|
| `primary` | `#3B82F6` | Azul metálico — botones principales, enlaces, acentos |
| `secondary` | `#FF6B35` | Naranja — acciones secundarias, llamadas a acción alternativa |
| `tertiary` | `#10B981` | Verde teal — éxito, rachas, logros, temporizadores |
| `background` | `#0F172A` | Fondo oscuro azulado (app bar, scaffold) |
| `surface` | `#1E293B` | Tarjetas, superficies elevadas |
| `error` | `#EF4444` | Errores, alertas |
| `gold` | `#F59E0B` | Recompensas, monedas, niveles |
| `white` | `#FFFFFF` | Texto principal |
| `grey` | `#B0B0B0` | Texto secundario |
| `darkGrey` | `#6B6B6B` | Texto terciario, deshabilitado |

## Combinación

- **Base**: Fondo oscuro `#0F172A` con superficies `#1E293B` para contraste sutil
- **Acento principal**: Azul metálico `#3B82F6` para CTAs, botones, iconos activos
- **Acento secundario**: Naranja `#FF6B35` para variación en cards, modos alternativos
- **Acento de éxito**: Verde teal `#10B981` para confirmaciones, progreso, timers de descanso
- **Recompensas**: Dorado `#F59E0B` para XP, niveles, trofeos

## Definición

Ver `app_pasos/lib/config/theme.dart` para la implementación completa.
